# Operationalising Otherness

This repository contains materials for our EADH 2026 conference abstract evaluating LLMs in the annotation of 'Otherness' in literary texts.

## Repository Contents
* **`curated_dataset.csv`**: Annotated dataset containing 100 literary passages with human labels, LLM predictions (GPT-5 and LLaMA 3), and our final curated consensus on Otherness.
* **`PROMPTS.md`**: The exact prompt configurations used to operationalise 'Otherness' for the models.
* **`curation_agreement.ipynb`**: Google Colab notebook containing the Python code for the project workflows.
